# 🌮 타코몰리(Tacomoly) 브랜드 웹사이트 개발 계획

## 1. 🎯 프로젝트 목표 (벤치마킹: 엽기떡볶이)
- **강력한 브랜드 아이덴티티 전달**: "맛있는 즐거움, 타코몰리"라는 브랜드 슬로건과 멕시칸 감성을 시각적으로 전달.
- **가맹점 모집 활성화**: 예비 창업자에게 수익성, 창업 절차, 비용 등 상세 정보를 제공하여 가맹 상담 유도.
- **주문 접근성 강화**: 웹사이트 방문 고객을 자사 주문 앱(`delivery-customer`)으로 자연스럽게 연결하여 수수료 절감.
- **고객 편의 제공**: 내 주변 매장 찾기, 신메뉴 및 이벤트 소식 전달.

## 2. 🗺️ 사이트맵 (Sitemap)

### 1. 메인 (Home)
- **Main Banner**: 신메뉴 프로모션, 브랜드 영상/이미지 슬라이드.
- **Quick Links**: 주문하기, 매장찾기, 메뉴소개, 프랜차이즈 바로가기.
- **Best Menu**: 시그니처 메뉴 슬라이드.
- **Brand Story 요약**: 타코몰리의 철학 한 줄 요약.
- **Instagram Feed**: SNS 연동 (선택 사항).

### 2. 메뉴 소개 (Menu)
- 카테고리별 메뉴 리스트 (타코, 부리또, 퀘사디아, 사이드/음료).
- 메뉴 상세 팝업/페이지 (이미지, 가격, 설명, 알레르기 정보, 맵기 단계).

### 3. 매장 찾기 (Store)
- **지도 연동**: Kakao 또는 Naver Map API 활용.
- **매장 검색**: 지역별 검색, 내 위치 기반 가까운 매장 찾기.
- **매장 리스트**: 매장명, 주소, 전화번호, 영업시간, 편의시설 아이콘.

### 4. 이벤트 & 소식 (Event & News)
- 진행 중인 이벤트 (썸네일 그리드).
- 공지사항 (게시판 형태).

### 5. 프랜차이즈 (Franchise)
- **경쟁력 소개**: 왜 타코몰리인가? (수익성, 차별화).
- **창업 절차**: 개설 문의부터 오픈까지 단계별 안내.
- **개설 비용**: 예상 투자 비용표.
- **가맹 상담 신청**: 이름, 연락처, 희망 지역 입력 폼.

### 6. 브랜드 스토리 (Brand Story)
- 브랜드 철학, BI 소개, 재료 이야기.

## 3. 🛠️ 기술 스택
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **Deployment**: Vercel (완료)
- **Maps**: Kakao Map API (예정)

## 3.1 🌍 환경 변수 (Environment Variables)
| 변수명 | 설명 | 비고 |
| :--- | :--- | :--- |
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase 프로젝트 URL | 필수 |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase Anon Key | 필수 |
| `NEXT_PUBLIC_TOSS_CLIENT_KEY` | 토스페이먼츠 Client Key | 필수 |
| `NEXT_PUBLIC_API_URL` | 백엔드 API URL | 필수 |
| `NEXT_PUBLIC_STORE_ID` | 매장 ID | 선택 (기본: store-1) |
