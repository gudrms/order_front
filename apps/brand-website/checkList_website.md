# 홈페이지 체크리스트
마지막 업데이트: 2026-05-02

## 현재 요약

- 브랜드 홈페이지 구조는 존재한다.
- MVP에서는 홈페이지가 배달앱의 매장 주문 URL로 보내는 방식이 가장 안전하다.
- 이후 홈페이지 직접 주문을 열면 `Order.source = HOMEPAGE`로 주문을 생성하고, 결제 완료 뒤 POS/알림 후처리는 백엔드 MQ가 담당한다.
- 홈페이지 프론트는 큐를 직접 구독하지 않고 REST API, 주문 상태 조회, 필요한 경우 Realtime/polling으로 확정된 상태를 표시한다.
