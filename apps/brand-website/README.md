# Brand Website

브랜드 마케팅 홈페이지

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **Rendering**: Static Site Generation (SSG) 중심
- **Styling**: TailwindCSS 4
- **SEO**: 최적화 적용

## 개발 서버 실행

```bash
pnpm dev
```

포트: http://localhost:3002

## 주요 기능

- 브랜드 소개
- 메뉴 소개
- 매장 위치
- 이벤트/프로모션
- 채용 정보

## 배포

정적 사이트로 빌드하여 Vercel에 배포

```bash
pnpm build
```
