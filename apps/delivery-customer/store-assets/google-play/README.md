# Google Play Store Assets

This directory contains Google Play Console listing assets for the delivery customer app.

These files are not app runtime assets and are intentionally kept outside `public/` so they are not served or bundled by the Next.js app.

Upload mapping:

- `icon-512x512.jpg`: app icon
- `feature-graphic-1024x500.jpg`: feature graphic
- `screenshots-phone/`: phone screenshots
- `screenshots-7-inch-tablet/`: 7-inch tablet screenshots
- `screenshots-10-inch-tablet/`: 10-inch tablet screenshots
