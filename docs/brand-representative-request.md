# 대표 확인 요청 문구

아래 문구는 브랜드 사이트 콘텐츠 확정을 위해 대표님께 카카오톡으로 보낼 초안입니다.

---

매형, 타코몰리 홈페이지에 들어갈 내용 몇 가지만 확인 부탁드려요.

1. 브랜드 스토리
- 타코몰리를 처음 시작하게 된 계기
- 왜 타코/멕시칸 메뉴를 선택했는지
- 타코몰리가 손님에게 주고 싶은 이미지나 약속
- 인천 중심으로 매장을 늘려온 과정에서 강조하고 싶은 점
- 홈페이지에 꼭 들어갔으면 하는 한 문장 슬로건

예시로는 이런 방향이에요.
“타코몰리는 부담 없이 즐길 수 있는 캐주얼 멕시칸 브랜드로, 신선한 재료와 대중적인 메뉴 구성으로 인천 지역에서 꾸준히 매장을 확장해왔습니다.”

2. 가맹/창업 페이지에 공개 가능한 정보
- 개설 비용을 홈페이지에 금액으로 공개해도 되는지
- 공개한다면 기준 평수와 항목별 금액
- 아직 공개가 부담스러우면 “상담 후 안내”로만 표기해도 되는지
- 평균 매출, 수익률 같은 숫자를 공개할 수 있는지
- 공개 가능한 숫자가 없다면 수익률 표현은 빼고 안정적인 운영/교육/메뉴 구성 위주로 적어도 되는지

3. 푸터 사업자 정보
- 상호
- 대표자명
- 사업자등록번호
- 본사 또는 사업장 주소
- 대표번호
- 고객 문의 이메일

확정된 내용만 사이트에 넣고, 애매한 숫자나 비용은 일단 “상담 후 안내”로 처리하려고 합니다.
