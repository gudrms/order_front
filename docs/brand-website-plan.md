# 타코몰리 브랜드 사이트 보강 기획안

> 방향: 현재 구현된 구조를 유지하고, 인천 중심 7개 매장 프랜차이즈로서의 신뢰와 전환 요소를 보강  
> 작성일: 2026-05-17

---

## 기획 방향

현재 사이트는 이미 `브랜드 / 메뉴 / 매장 / 가맹 문의 / 개인정보처리방침` 구조로 개발되어 있다.  
이번 기획은 새 페이지를 크게 늘리기보다, 기존 화면 안에서 부족한 정보를 채워 브랜드 신뢰도와 주문/가맹 전환을 높이는 것을 목표로 한다.

- **목적 1**: 처음 방문한 사람이 타코몰리가 어떤 브랜드인지 바로 이해
- **목적 2**: 매장 찾기와 주문 CTA를 강화해 실제 주문으로 연결
- **목적 3**: 인천 중심 7개 매장 운영 사실을 활용해 가맹 문의 신뢰 확보
- **목적 4**: 임시 카피, 임시 이미지, 미확정 수치를 실제 운영 정보 기준으로 정리

---

## 현재 유지할 사이트맵

```
tacomole.kr
│
├── 홈 (/)
├── 브랜드 소개 (/brand)
├── 메뉴 소개 (/menu)
├── 매장 찾기 (/store)
├── 가맹 문의 (/franchise)
└── 개인정보처리방침 (/privacy)
```

### 이번 단계에서 제외 또는 후순위

| 항목 | 판단 | 이유 |
|---|---|---|
| 회사 페이지 `/company` | 후순위 | 현재 7개 매장 규모에서는 별도 회사 페이지보다 브랜드/푸터 정보 보강이 자연스러움 |
| BI 탭 | 후순위 | 고객/가맹 희망자에게 우선순위 낮음. 로고 가이드 자료가 준비되면 확장 |
| 인테리어 탭 | 후순위 | 매장 사진이 충분히 준비된 뒤 브랜드 페이지 섹션으로 추가 가능 |
| 공지/이벤트 목록 페이지 | 후순위 | 운영 부담이 생기므로 홈에 작은 소식 영역만 선택적으로 추가 |

## 구현 현황 기준

| 페이지 | 현재 상태 | 보강 방향 |
|---|---|---|
| 홈 `/` | 히어로, 대표 메뉴, 가맹 섹션 구현 | 인천 7개 매장 신뢰 문구, 매장 찾기/주문 CTA 보강 |
| 브랜드 `/brand` | 브랜드 스토리, 가치 섹션 구현 | 실제 브랜드 스토리로 교체, 임시 이미지/이모지 제거 |
| 메뉴 `/menu` | 구현 완료 | 실제 메뉴 사진/가격/뱃지 검수 |
| 매장 `/store` | 카카오 지도, 매장 목록, 주문 링크 구현 | 홈에서 이 페이지로 이어지는 CTA 강화 |
| 가맹 `/franchise` | 장점, 비용 요약, 상담 폼 구현 | 미확정 수치 검증, 창업 절차 보강, 개인정보 동의 추가 |
| 개인정보 `/privacy` | 구현 완료 | 푸터 링크 연결 상태 확인 |

## 페이지별 보강 기획

---

### 홈 (/)

현재 구조:

```
1. Hero
2. MenuSection
3. FranchiseSection
```

권장 보강 구조:

```
1. Hero
2. Brand Trust Strip       신규 또는 Hero 하단 보강
3. MenuSection             기존 유지
4. Store Order Section     신규
5. FranchiseSection        기존 보강
```

#### 1. Hero 보강

현재 카피는 유지 가능하나, 지역성과 실매장 운영감을 조금 더 드러낸다.

권장 문구:

- 헤딩: `맛있는 즐거움, 타코몰리입니다.`
- 서브: `인천에서 시작해 7개 매장으로 확장한 캐주얼 멕시칸 브랜드`
- CTA:
  - `메뉴 보기`
  - `지금 주문하기`
  - `매장 찾기`

확인 필요:

- 히어로 배경 이미지는 현재 외부 Unsplash 이미지 사용 중
- 실제 메뉴 또는 매장 사진 확보 시 교체 권장

#### 2. Brand Trust Strip 추가

히어로 아래에 짧은 신뢰 지표를 추가한다.

예시:

| 항목 | 문구 |
|---|---|
| 운영 지역 | 인천 중심 운영 |
| 매장 수 | 7개 매장 운영 |
| 주문 연결 | 매장별 바로 주문 |
| 창업 상담 | 가맹 상담 접수 가능 |

주의:

- `7개 매장`은 실제 운영 매장 수와 일치해야 함
- 수익률, 매출액 등 민감 수치는 검증 전까지 사용하지 않음

#### 3. MenuSection 유지

현재 대표 메뉴 4개 카드 구조는 좋다.

보강할 점:

- 실제 대표 메뉴명으로 교체
- 실제 메뉴 사진으로 교체
- 가격 표시 여부 결정
- `BEST`, `NEW` 뱃지는 실제 기준이 있을 때만 사용

#### 4. Store Order Section 추가

홈에서 `/store`로 넘어가는 전환 지점이 현재 약하다.  
매장 찾기와 주문은 이 사이트의 핵심 행동이므로 홈에 별도 섹션을 둔다.

구성:

- 섹션 제목: `가까운 타코몰리 매장을 찾아보세요`
- 설명: `인천 주요 지역에서 타코몰리를 만나보실 수 있습니다.`
- 매장 카드 3개 또는 대표 지역 카드
- CTA:
  - `전체 매장 보기`
  - `지금 주문하기`

매장 카드 예시:

| 표시 항목 | 내용 |
|---|---|
| 매장명 | 실제 매장명 |
| 지역 | 인천 ○○구 |
| 주문 버튼 | 해당 매장 주문 링크 |

#### 5. FranchiseSection 보강

현재 `7호점`, `30% 평균 수익률` 지표가 있다.  
이 중 `30% 평균 수익률`은 확정 자료가 없다면 삭제하거나 완곡한 표현으로 교체한다.

권장:

- `7호점 돌파` → 실제 매장 수와 일치하면 유지
- `30% 평균 수익률` → 검증 전 삭제 또는 `운영 효율을 고려한 메뉴 구성`으로 변경
- 카피는 과장보다 안정감 중심으로 정리

권장 문구:

```text
인천에서 검증해온 캐주얼 멕시칸 브랜드.
타코몰리는 조리 효율, 배달 적합성, 대중적인 메뉴 구성을 바탕으로
예비 점주님의 창업 상담을 돕습니다.
```

### 브랜드 소개 (/brand)

현재 구성:

```
1. Hero
2. Philosophy Section
3. Values Section
```

#### 보강 방향

브랜드 페이지는 탭으로 확장하지 않고, 현재 단일 페이지 구조를 유지한다.  
대신 임시성 있는 스토리와 이미지를 실제 브랜드에 맞게 교체한다.

#### 교체 필요 항목

| 현재 요소 | 문제 | 보강 방향 |
|---|---|---|
| `2024년 작은 푸드트럭에서 시작` | 실제가 아니면 신뢰 리스크 | 실제 창업 배경으로 교체 |
| 이모지 플레이스홀더 | 공식 브랜드 사이트 느낌 약함 | 실제 매장/메뉴 사진으로 교체 |
| `멕시코의 태양` 등 추상 카피 | 차별점이 흐림 | 인천 기반, 대중적 멕시칸, 배달/매장 운영 강점 반영 |

#### 권장 섹션

```text
1. OUR STORY
2. 인천에서 시작한 타코몰리
3. 우리가 지키는 약속 3가지
4. 매장과 메뉴 사진
```

#### 차별화 문구 후보

- 매일 부담 없이 즐기는 캐주얼 멕시칸
- 매장과 배달에 모두 적합한 메뉴 구성
- 신선한 재료와 균일한 조리 기준
- 인천 주요 상권에서 운영 중인 브랜드

---

### 메뉴 소개 (/menu)

현재 구현을 유지한다.

확인할 것:

- 실제 메뉴명과 가격 반영
- 품절/미판매 메뉴 노출 여부
- 대표 메뉴와 전체 메뉴의 기준 일치
- 메뉴 이미지가 실제 제품과 크게 다르지 않은지 확인

개발 변경 우선순위는 낮지만, 사진 품질은 사이트 신뢰도에 직접 영향을 준다.

---

### 매장 찾기 (/store)

현재 구현을 유지한다.

이미 잘 맞는 점:

- 카카오 지도
- 매장 목록
- 검색
- 현재 위치 기반 정렬
- 매장별 주문 링크

보강할 점:

- 홈에서 `/store` 진입 CTA 강화
- 매장 수가 7개라면 전체 매장이 정확히 노출되는지 확인
- 각 매장 주문 URL이 실제 배달 주문 페이지와 연결되는지 확인

### 가맹 문의 (/franchise)

현재 구성:

```
1. FRANCHISE Hero
2. Why TACO MOLE?
3. 가맹 개설 비용 요약
4. 가맹 상담 신청 폼
```

#### 핵심 보강 방향

가맹 페이지는 사이트에서 가장 민감한 페이지다.  
따라서 수익률, 비용, 창업 절차, 개인정보 수집 동의는 실제 기준으로 정리해야 한다.

#### 1. 숫자 검증

현재 또는 향후 표시할 숫자는 반드시 확인한다.

| 항목 | 처리 |
|---|---|
| 매장 수 | 실제 운영 매장 기준으로 표시 |
| 평균 수익률 | 근거 자료 없으면 삭제 권장 |
| 개설 비용 | 확정 전이면 `상담 후 안내` 또는 `표준 모델 기준`으로 표시 |
| 교육 기간 | 실제 교육 기준 확인 후 표시 |

#### 2. 창업 절차 추가

탭을 만들지 않고 현재 페이지 안에 섹션으로 추가하는 것을 권장한다.

```text
STEP 01  가맹 상담
STEP 02  상권 및 후보지 검토
STEP 03  정보공개서 제공
STEP 04  가맹 계약
STEP 05  인테리어 및 설비 준비
STEP 06  본사 교육
STEP 07  오픈 준비
STEP 08  매장 오픈 및 운영 지원
```

#### 3. 비용 정보 표시 방식

실제 비용이 확정되어 있으면 표로 표시한다.  
확정 전이라면 구체 금액보다 안내형으로 둔다.

권장 문구:

```text
개설 비용은 점포 면적, 상권, 인테리어 조건에 따라 달라질 수 있습니다.
상담 접수 후 표준 모델 기준의 상세 비용을 안내드립니다.
```

#### 4. 문의 폼 보강

현재 필드:

- 이름
- 연락처
- 이메일
- 희망 창업 지역
- 문의 내용

추가 권장:

- 개인정보 수집 및 이용 동의 체크박스
- 연락 가능 시간대
- 창업 희망 시기

### 푸터

현재 푸터는 정보가 부족하다.  
회사 페이지를 만들지 않는 대신 푸터에 기본 사업자 정보를 보강한다.

필수 항목:

| 항목 | 상태 |
|---|---|
| 상호 | 확정 필요 |
| 대표자 | 확정 필요 |
| 사업자등록번호 | 확정 필요 |
| 대표번호 | 확정 필요 |
| 이메일 | 현재 `tacomole26@gmail.com` 사용 중 |
| 주소 | 확정 필요 |
| 개인정보처리방침 링크 | `/privacy` 연결 필요 |

선택 항목:

- 인스타그램
- 네이버 블로그
- 카카오톡 채널
- 유튜브

## 개발 우선순위

### Phase 1 — 현재 구조 보강

| 순서 | 작업 | 이유 |
|---|---|---|
| 1 | 홈에 인천 7개 매장 신뢰 문구 추가 | 브랜드 실체감 강화 |
| 2 | 홈에 매장 찾기/주문 CTA 섹션 추가 | 주문 전환 강화 |
| 3 | 가맹 섹션의 수익률/비용 수치 검증 또는 완곡 처리 | 신뢰 리스크 방지 |
| 4 | 브랜드 페이지 임시 스토리/이미지 교체 | 공식 사이트 완성도 개선 |
| 5 | 푸터 사업자 정보 및 개인정보 링크 보강 | 기본 신뢰 정보 확보 |

### Phase 2 — 콘텐츠 확보 후

| 순서 | 작업 | 조건 |
|---|---|---|
| 6 | 실제 메뉴/매장 사진 교체 | 사진 확보 |
| 7 | 가맹 창업 절차 섹션 추가 | 절차 확정 |
| 8 | 가맹 비용 안내 정리 | 비용 기준 확정 |
| 9 | 공지/이벤트 작은 섹션 추가 | 운영할 소식 2~3개 확보 |

### Phase 3 — 확장 후보

| 작업 | 조건 |
|---|---|
| `/company` 페이지 추가 | 회사 소개, 연혁, 본사 정보가 정리된 뒤 |
| BI 섹션 추가 | 로고 가이드 자료 확보 |
| 인테리어 갤러리 추가 | 매장 내외부 사진 6~8장 확보 |
| 공지/이벤트 목록 페이지 | 지속 운영 가능할 때 |

## 결론

현재 개발 구조는 인천 중심 7개 매장 프랜차이즈 단계에 적합하다.  
따라서 새 사이트맵으로 크게 바꾸기보다, 현재 페이지를 유지한 채 홈과 가맹 페이지의 전환력을 높이고 브랜드/푸터의 신뢰 정보를 채우는 방식이 가장 현실적이다.
