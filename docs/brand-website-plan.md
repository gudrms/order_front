## 1. 홈 (/)

### 헤더
- 로고 (타코몰리)
- GNB 메뉴
- SNS 링크: 인스타그램, 유튜브, 카카오톡, 블로그

### 메인 히어로
- 슬로건: `"진짜 멕시칸, 진짜 맛"` *(작성 필요)*
- 서브카피: `"매일 먹고 싶은 타코, 타코몰리"`
- 배경: 대표 메뉴 영상 또는 이미지

### 섹션 1 — 대표 메뉴
- 타코 / 부리또 / 케사디아 등 카테고리별 대표 메뉴 썸네일
- `"메뉴 보기"` CTA → `/메뉴`

### 섹션 2 — 브랜드 한줄 소개
- 짧은 브랜드 철학 카피
- `"브랜드 스토리"` CTA → `/브랜드`

### 섹션 3 — 매장 찾기
- 지도 미니 프리뷰 또는 대표 매장 이미지
- `"가까운 매장 찾기"` CTA → `/매장`
- `"지금 주문하기"` CTA → delivery.tacomole.kr

### 섹션 4 — 이벤트/공지
- 최근 이벤트 2~3개 카드
- 최근 공지 목록

### 섹션 5 — 창업 문의
- 배경 이미지 + 창업 유도 카피
- `"창업 문의하기"` CTA → `/창업`

## 2. 브랜드 (/브랜드)

### 탭 구성: 브랜드 소개 | BI | 인테리어

---

### 2-1. 브랜드 소개

**섹션 1 — 핵심 메시지**
```
"TACOMOLE IS"
```
- 차별화 요소 4가지 카드 형식으로 나열 (보끔당의 "BOKKEUMDANG IS" 구조 참고)
  - 예: `신선한 재료` / `정통 레시피` / `합리적인 가격` / `빠른 서비스`

**섹션 2 — 브랜드 스토리**
- 헤딩: `"타코가 맛있는 집, 타코몰리입니다"`
- 본문 카피 *(작성 필요)*
- 브랜드 창업 배경 / 철학

**섹션 3 — 차별화 포인트**
- 보끔당 참고: `국내산 재료`, `비법 소스`, `개별 조리`
- 타코몰리 버전 *(작성 필요)*:
  - 예: `매일 들어오는 신선 재료`
  - 예: `직접 개발한 타코 소스`
  - 예: `주문 즉시 조리`

---

### 2-2. BI (Brand Identity)

- **네이밍 설명**: 타코몰리 = TACO + MOLE *(과카몰리, 멕시칸 소스)*
- **로고 가이드**
  - 가로형 로고
  - 세로형 로고
- **컬러 시스템**
  - MAIN: `brand-yellow` (#FACC15 계열)
  - SUB: `brand-black`, `brand-green`
- **폰트 가이드** *(있다면)*

---

### 2-3. 인테리어

- 매장 내외부 사진 갤러리 (6~8장)
- 카피: `"밝고 활기찬 멕시칸 감성의 공간"`
- 각 매장 인테리어 컨셉 소개

## 4. 매장 (/매장)

### 검색 UI
- 지역명 입력 검색
- 매장명 검색
- `"내 주변 매장 찾기"` GPS 버튼

| 전화번호 | 02-XXX-XXXX |

| 단계 | 항목 | 내용 |
|---|---|---|
| 1 | 가맹 상담 | 지역 문의, 면담 일정, 구비서류 안내 |
| 2 | 가맹 후보자 선정 | 본사 내방 면담, 최종 선정 |
| 3 | 정보공개서 제공 | 정보공개서 및 계약서 사전 제공 |
| 4 | 입점 후보지 선택 | 임대 물건 탐색 및 협의 |
| 5 | 가맹계약 체결 | 계약서 작성, 예치금 납부 |

| 7 | 본사 교육 | 운영 / 조리 / CS 교육 이수 |

## 현재 구현 현황

| 페이지 | 구현 상태 | 비고 |
|---|---|---|
| 홈 (`/`) | ✅ 완료 | 히어로, 메뉴, 브랜드, 매장, 가맹 섹션 |
| 브랜드 소개 (`/brand`) | ✅ 완료 | 기본 구성 |
| 메뉴 (`/menu`) | ✅ 완료 | 카테고리별 메뉴 표시 |
| 매장 찾기 (`/store`) | ✅ 완료 | 카카오 지도 + 목록 분할 레이아웃, 주문 링크 |
| 가맹 문의 (`/franchise`) | ✅ 완료 | 문의 폼 (DB 저장) |
| 개인정보처리방침 (`/privacy`) | ✅ 완료 | Play Store 심사용 |
| **브랜드 > BI** | ❌ 미구현 | 로고/컬러 가이드 페이지 |
| **브랜드 > 인테리어** | ❌ 미구현 | 매장 사진 갤러리 |
| **창업 절차** | ❌ 미구현 | 단계별 창업 프로세스 |
| **개설 비용** | ❌ 미구현 | 비용 테이블 |
| **회사 > 인사말** | ❌ 미구현 | CEO 메시지 |
| **회사 > 연혁** | ❌ 미구현 | 타임라인 |
| **회사 > 오시는 길** | ❌ 미구현 | 본사 지도 |
| **이벤트/공지** | ❌ 미구현 | 이벤트 목록, 공지사항 |
