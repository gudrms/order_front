> 작성일: 2026-05-17

---

## 기획 방향

- **목적 1**: 브랜드 인지 — 처음 방문한 사람이 "어떤 브랜드인지" 바로 파악
- **목적 2**: 매장 찾기 → 배달앱 주문 전환
- **목적 3**: 가맹 문의 유도
- **보조**: 홈에 공지/이벤트 섹션 1개 — 신메뉴·가맹 소식으로 사이트가 살아있어 보이게

## 페이지별 상세 기획

---

### 홈 (/)

#### 레이아웃 순서

```
[헤더 — 로고 + GNB + SNS 링크]

1. 히어로
2. 브랜드 한줄 소개
3. 대표 메뉴
4. 매장 찾기 + 주문 CTA      ← 핵심 전환 지점
5. 공지 / 이벤트             ← 엽기떡볶이에서 차용, 1개 섹션만
6. 창업 문의 배너

[푸터]
```

#### 섹션별 상세

**1. 히어로**
- 전체 너비 배경 이미지 또는 영상
- 헤딩: 슬로건 (예: `"진짜 멕시칸, 진짜 맛"`) ← 텍스트 확정 필요
- 서브: 한 줄 설명
- CTA 버튼 2개: `메뉴 보기` / `가까운 매장 찾기`

**2. 브랜드 한줄 소개**
- 배경 색상 블록 (brand-yellow 또는 brand-black)
- 짧은 브랜드 철학 한 문장
- `브랜드 스토리 →` 링크

**3. 대표 메뉴**
- 카테고리 대표 메뉴 3~4개 카드 (이미지 + 메뉴명 + 가격)
- `전체 메뉴 보기 →` 링크

**4. 매장 찾기 + 주문 CTA** ← 가장 중요한 섹션
- 미니 지도 또는 매장 카드 2~3개
- 각 매장에 `지금 주문하기` 버튼 → `delivery.tacomole.kr/store/{id}/menu`
- `전체 매장 보기 →` 링크

**5. 공지 / 이벤트**
- 최근 공지 또는 이벤트 2~3개 카드
- 제목 + 날짜 + 짧은 설명
- `더 보기 →` (추후 목록 페이지 추가 가능)
- *운영: 관리자 페이지에서 등록 또는 정적 데이터로 시작*

**6. 창업 문의 배너**
- 배경 이미지 + 카피: `"타코몰리와 함께 창업하세요"`
- CTA: `창업 문의하기` → `/franchise`

#### 헤더
- 로고
- GNB: 브랜드 / 메뉴 / 매장 / 창업 / 회사
- SNS 아이콘: 인스타그램, 유튜브, 카카오톡, 블로그

#### 푸터
| 항목 | 비고 |
|---|---|
| 상호 | 확정 필요 |
| 사업자등록번호 | 확정 필요 |
| 대표자 | 확정 필요 |
| 대표번호 | 확정 필요 |
| 고객센터 | 확정 필요 |
| 주소 | 확정 필요 |
| 이메일 | 확정 필요 |

- 이용약관 / 개인정보처리방침 링크
- SNS 링크
- COPYRIGHT

### 브랜드 (/brand)

탭 3개: **브랜드 소개** | **BI** | **인테리어**

#### 탭 1 — 브랜드 소개

```
[페이지 헤더: "BRAND"]

섹션 1: 핵심 메시지
  - 대형 헤딩: "TACOMOLE IS"
  - 차별화 포인트 4개 카드 (아이콘 + 한 줄 설명)
    예: 신선한 재료 / 정통 레시피 / 합리적 가격 / 빠른 조리

섹션 2: 브랜드 스토리
  - 헤딩: "타코가 맛있는 집, 타코몰리입니다"
  - 본문: 브랜드 창업 배경 + 철학 (3~4문단)  ← 텍스트 확정 필요
  - 이미지: 대표 매장 사진

섹션 3: 차별화 3가지  (보끔당 참고)
  - "국내산 신선 재료" / "직접 개발한 소스" / "주문 즉시 조리"
    ← 실제 문구 확정 필요
```

#### 탭 2 — BI

```
섹션 1: 네이밍
  - TACO + MOLE(과카몰리) 설명

섹션 2: 로고 가이드
  - 가로형 / 세로형 로고 이미지

섹션 3: 컬러 시스템
  - MAIN: Yellow (#FACC15 계열)
  - SUB: Black, Green
  - 컬러 칩 + 코드 표시

섹션 4: 폰트 (있다면)
```

#### 탭 3 — 인테리어

```
섹션 1: 카피
  - "밝고 활기찬 멕시칸 감성의 공간"

섹션 2: 갤러리
  - 매장 내외부 사진 6~8장 (그리드 레이아웃)
  - 클릭 시 라이트박스 확대

섹션 3: 인테리어 컨셉 설명
  - 1~2문단
```

### 창업 (/franchise)

탭 2개: **창업 절차** | **개설 비용**  
(기존 가맹 문의 폼은 탭 하단 공통 배치)

#### 탭 1 — 창업 절차

```
[페이지 헤더: "FRANCHISE"]

섹션 1: 창업 절차 타임라인 (9단계)
  STEP 01  가맹 상담       지역 문의, 면담 일정, 구비서류 안내
  STEP 02  후보자 선정     본사 내방 면담, 최종 선정
  STEP 03  정보공개서 제공  계약서 사전 제공 (법정 14일)
  STEP 04  입점 후보지     임대 물건 탐색 및 협의
  STEP 05  가맹계약 체결   계약서 작성, 예치금 납부
  STEP 06  인테리어 시공   브랜드 맞춤 공사 (~3~4주)
  STEP 07  본사 교육       운영 / 조리 / CS 교육 이수
  STEP 08  설비·식자재 입고 주방설비 및 초도 물품
  STEP 09  가맹점 오픈    오픈 바이저 지원, 월 정기 점검

섹션 2: 가맹 문의 폼
  - 이름 / 연락처 / 희망 지역 / 문의 내용
  - 개인정보 수집 동의 체크박스
  - "문의 접수하기" 버튼
  ← 현재 DB 저장 구현 완료
```

#### 탭 2 — 개설 비용

```
섹션 1: 비용 테이블
  기준: XX평 / 점포 별도

  | 항목        | 금액       |
  |------------|-----------|
  | 가맹비      | -         |
  | 교육비      | -         |
  | 인테리어    | -         |
  | 간판·사인물 | -         |
  | 가구       | -         |
  | 주방설비    | -         |
  | 주방기물    | -         |
  | POS        | -         |
  | 초도식자재  | -         |
  | 합계       | - 원      |
  ← 실제 비용 확정 필요

섹션 2: 주의사항 / 안내 문구
  - 점포 임차 비용(보증금·권리금) 별도
  - 상기 금액은 표준 견적이며 면적·지역에 따라 상이

섹션 3: 가맹 문의 폼 (탭 1과 동일)
```

### 회사 (/company)

탭 4개: **인사말** | **회사 소개** | **연혁** | **오시는 길**

#### 탭 1 — 인사말

```
섹션: CEO 메시지
  - 헤딩: "대한민국 타코 문화의 기준이 되겠습니다" ← 확정 필요
  - 본문: 3~4문단 브랜드 가치 + 비전
  - 대표자 이름 + 직함
  - 대표자 사진 (선택)
```

#### 탭 2 — 회사 소개

```
섹션 1: 핵심 약속
  - "믿고 먹을 수 있는 신선하고 맛있는 타코"

섹션 2: 운영 철학
  - 위생 관리 기준
  - 조리 매뉴얼 준수
  - 변함없는 맛에 대한 약속

섹션 3: 수상·인증 (있다면)
  - HACCP 인증 등
```

#### 탭 3 — 연혁

```
타임라인 (세로 형태)

  20XX  타코몰리 1호점 오픈
  20XX  2호점 오픈
  20XX  배달앱 출시
  20XX  (추가 예정)
  ← 실제 연혁 확정 필요
```

#### 탭 4 — 오시는 길

```
섹션 1: 본사 정보
  - 주소 / 전화 / 이메일

섹션 2: 지도
  - 카카오 지도 임베드

섹션 3: 교통편 안내 (선택)
```
