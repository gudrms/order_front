## 1. 💡 왜 이 프로젝트인가요? (핵심 요약)

### "매달 나가는 고정비, 이제 그만!" 💸
시중의 테이블 오더(T오더 등)는 **테이블당 월 22,000원**의 이용료를 받습니다.
테이블이 20개라면? **3년에 1,500만원**이 넘는 돈이 고정비로 사라집니다.

**이 프로젝트는 '월 이용료 0원'입니다.**
초기 태블릿 구매 비용만 투자하면, 평생 무료로 사용할 수 있는 **자체 소유 시스템**입니다.

## 4. ✨ 우리만의 차별점 (Why Us?)

### 1️⃣ 내 가게에 딱 맞는 '커스텀 디자인' 🎨
- **T오더:** 정해진 디자인 틀 안에서만 로고 정도 변경 가능.
- **우리:** 우리 가게의 인테리어, 브랜드 컬러, 폰트에 맞춰 **100% 맞춤 디자인** 가능.
  - "우리 가게는 고급스러운 블랙 테마로 하고 싶어요" -> **가능!**
  - "메뉴 사진을 꽉 차게 보여주고 싶어요" -> **가능!**

### 2️⃣ 기능의 무한한 확장성 🚀
- **T오더:** 본사가 만들어준 기능만 사용 가능.
- **우리:** 필요한 기능은 언제든 개발해서 붙일 수 있음.
  - "단골 손님 생일 쿠폰을 보내주고 싶어요" -> **개발 가능!**
  - "우리 가게만의 룰렛 게임 이벤트를 넣고 싶어요" -> **개발 가능!**

### 3️⃣ 데이터의 주권 🔐
- **T오더:** 고객 데이터와 주문 데이터가 플랫폼사에 귀속됨.
- **우리:** 모든 데이터가 **사장님 소유**.
  - 어떤 메뉴가 언제 잘 팔리는지, 어떤 고객이 자주 오는지 직접 분석하고 마케팅에 활용 가능.

## 4. 📋 주요 기능 소개

### 🍽️ 고객용 태블릿 (테이블 오더)
**"직원을 부르지 않아도, 원하는 대로 주문!"**

- **📖 스마트 메뉴판**: 고화질 사진과 상세 설명으로 식욕 자극
- **🛒 장바구니 & 옵션**: "맵게", "오이 빼고" 등 복잡한 옵션도 터치 한 번으로
- **🔔 직원 호출**: "물 주세요", "앞치마 주세요" 등 버튼으로 해결 (직원 동선 최소화)
- **💸 더치페이 계산기**: (예정) 친구들과 편하게 계산 가능

### 🧑‍🍳 주방/카운터 시스템 (개발 예정)
**"주문 누락 제로! 주방의 평화"**

- **📢 실시간 주문 알림**: 주문 즉시 주방 모니터에 "띵동!"
- **🖨️ 영수증 자동 출력**: OKPOS 등 기존 포스기와 완벽 연동
- **📊 매출 리포트**: 오늘 얼마나 팔았는지 한눈에 확인
