```text
+--------------------------------------------------+
|                                           [ X ]  |
|  +--------------------------------------------+  |
|  |                                            |  |
|  |             ( 큰 메뉴 이미지 )             |  |

|  **까르보나라**                                  |
|  진한 크림 소스와 베이컨의 조화                  |

|                                                  |

|  **옵션 선택**                                   |
|  [ ] 맵게 (+500원)                               |
|  [ ] 면 추가 (+2,000원)                          |

```
