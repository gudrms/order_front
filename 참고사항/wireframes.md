# 📱 UI/UX Wireframes & User Flow

## 1. 🧑‍🤝‍🧑 고객용 태블릿 (Customer Tablet)

### 1.1 메인 화면 (Home)
- **Layout**:
    - **Top Bar**: 로고, 테이블 번호, 직원 호출 버튼
    - **Left Sidebar**: 카테고리 리스트 (메인, 사이드, 음료, 주류)
    - **Center Grid**: 선택된 카테고리의 메뉴 카드 리스트 (이미지, 이름, 가격)
    - **Bottom Bar**: 장바구니 요약 (총 수량, 총 금액), '주문하기' 버튼

### 1.2 메뉴 상세 팝업 (Menu Detail Modal)
- **Trigger**: 메인 화면에서 메뉴 카드 터치 시
- **Content**:
    - 큰 메뉴 이미지
    - 메뉴 이름 및 상세 설명
    - **옵션 선택**: 맵기 조절 (라디오 버튼), 토핑 추가 (체크박스)
    - **수량 조절**: -, + 버튼 및 숫자 입력
    - **Action**: '담기' 버튼, '취소' 버튼

### 1.3 장바구니 (Cart Drawer/Modal)
- **Trigger**: 하단 바 '주문하기' 영역 터치 시 확장
- **Content**:
    - 담은 메뉴 리스트 (메뉴명, 옵션, 수량, 가격)
    - 각 항목별 수량 조절 및 삭제 버튼
    - **Total**: 최종 결제 예정 금액
    - **Action**: '주문 전송' 버튼 (클릭 시 "주문하시겠습니까?" 컨펌 팝업)

### 1.4 주문 완료 및 내역 (Order History)
- **Trigger**: 주문 전송 성공 시
- **Content**:
    - "주문이 접수되었습니다" 메시지
    - 현재 테이블의 총 주문 내역 리스트
    - 각 메뉴별 상태 (접수됨 / 조리중 / 서빙완료)
    - **Total**: 현재까지의 총 식사 금액

## 2. 👨‍🍳 주방/홀 직원용 (Kitchen/POS)

### 2.1 주문 접수 현황판 (Order Dashboard)
- **Layout**:
    - **Grid Layout**: 각 주문 건이 카드 형태로 시간순 나열
- **Order Card**:
    - **Header**: 테이블 번호, 경과 시간 (5분 지남 등 색상 표시)
    - **Body**: 주문 메뉴 리스트, 옵션 상세
    - **Footer**: '조리 시작', '서빙 완료' 버튼
- **Alert**: 신규 주문 도착 시 화면 깜빡임 및 알림음 재생

### 2.2 테이블 현황 (Table Status)
- **Layout**: 매장 테이블 배치도와 유사한 그리드
- **Table Item**:
    - 테이블 번호
    - 상태 아이콘 (공석/식사중/치워야함)
    - 현재 이용 시간

---

## 3. 🛠️ 관리자 (Admin)

### 3.1 메뉴 관리 (Menu Management)
- **List View**: 전체 메뉴 리스트 (이미지, 이름, 가격, 카테고리, 품절여부)
- **Edit Form**: 메뉴 등록/수정 모달 (이미지 업로드, 옵션 그룹 설정)
- **Quick Action**: 리스트에서 '품절' 토글 스위치로 즉시 품절 처리

### 3.2 매출 통계 (Sales Analytics)
- **Chart**: 일주일간 매출 추이 (Line Chart)
- **Ranking**: 인기 메뉴 Top 5 (Bar Chart)
- **Table**: 일자별 상세 매출 내역 엑셀 다운로드
